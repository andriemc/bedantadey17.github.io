Cosmium - The All-purpose Discord Bot Add to Server

It's time to ditch MEE6 and Dyno.

I know how hard it is to find a bot which actually feels vanilla-friendly. With Discord's integration of slash commands, many bots still like to use stupid prefixes and duplicate commands (like kick, ban) which Discord already fulfills the need of.

Full of utility and fun commands; and smooth compatibility with Discord's slash commands -- this is the only bot you will ever need. I will never add commands which Discord already has.


Utility Commands
  • /avatar - Displays the avatar of a user.
  • /help - Displays available commands.
  • /ping - Check the bot's latency.
  • /purge - Deletes a specified number of recent messages.
  • /server - Get information about the server.
  • /slowmode - Set the slowmode of the channel.
  • /user - Displays information about a user.
